# FairSight AI

FairSight AI is an AI-powered system for detecting, explaining, and mitigating bias in machine learning models.

## Features

* Bias detection using statistical metrics
* Risk classification (Low / Moderate / High)
* Visual insights with charts
* AI-powered explanations using Gemini (Vertex AI)
* Exportable bias reports

## Deployment

Deployed on Google Cloud Run with Gemini integration via Vertex AI.

## Tech Stack

* Python
* Streamlit
* Google Cloud (Cloud Run, Vertex AI)
* Pandas, Matplotlib

## Live Demo

https://fairsight-ai-695622439296.us-central1.run.app

## Problem

AI systems can produce biased outcomes. FairSight AI helps identify and explain these biases to ensure fair decision-making.

## Solution

We provide a real-time system that:

* Detects bias
* Explains it in human terms
* Suggests actionable improvements
